# Pokemon GO IV calculator and database
## Usage

### First set executable
    chmod +x pgo

### calculate IVs
    pgo calc [pokemon name] [CP] [Health] [Dust to power up] [phrase] [strongest feature] [perfect iv]

### Phrases
    - amazing
    - strong
    - decent
    - bad
    - all

### Strongest feature
    - atk
    - def
    - stam

### If pokemon has at least 1 perfect IV
    - wow
